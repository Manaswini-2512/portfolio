🌍 Trawell

A tourism and handicraft e-commerce platform that bridges travellers with authentic local experiences and artisan products — built end-to-end on the MERN stack.

TypeFull Stack Web App
RoleSolo Developer
StackMERN

Overview

Trawell was built to make local tourism more accessible and give artisans a digital storefront. Users can discover tourism destinations through curated guides and purchase handicraft products directly — all in one seamless platform.

The project covers the entire product lifecycle: user authentication, destination browsing, product listings, and cart management — making it feel like a genuine production-grade application.

My Role & Contribution

🖥️ Frontend

  • Designed and built all React components from scratch
  • Implemented fully responsive layouts for mobile and desktop
  • Built destination browsing and product listing pages
  • Handled client-side routing using React Router

⚙️ Backend

  • Architected the REST API with Express.js
  • Designed MongoDB schemas for users, destinations, and products
  • Implemented JWT-based authentication (login/signup)
  • Connected frontend to backend via Axios API calls

Key Features

  • Secure user authentication and session management with JWT
  • Browse and explore tourism destinations with rich detail pages
  • Handicraft product listings with descriptions and images
  • Responsive UI that works across all screen sizes
  • Clean RESTful API with separation of concerns across routes and controllers

Challenges & How I Solved Them

🔐 Auth State Across Components

  • Keeping login state in sync across deeply nested components was tricky
  • Lifted auth state to the App root and passed it via context, avoiding prop drilling
  • Stored JWT in localStorage and attached it to every protected API request via Axios interceptors

📦 Data Modelling

  • Deciding how to relate users, destinations, and products in a NoSQL DB required planning
  • Used referenced documents for users and embedded sub-documents for product metadata to balance performance and flexibility
